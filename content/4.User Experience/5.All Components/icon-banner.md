# Icon Banner

![Icon Banner](/img/Icon_Banner.png)

**Use case**<br>
When you want to emphasize a link to another page or action.

**Limitations**<br>
There can only be one CTA. 

**What’s required?**<br>
The section heading and icon must be included.

**What’s optional?**<br>
CTA & image
