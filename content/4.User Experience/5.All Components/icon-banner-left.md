# Icon Banner – Left

![Icon Banner – Left](/img/Icon_Banner_Left.png)

**Use case**<br>
When you want to emphasize a link to another page or action with the image on the left

**Limitations**<br>
There can only be one CTA. 

**What’s required?**<br>
The section heading and icon must be included.

**What’s optional?**<br>
CTA & image
